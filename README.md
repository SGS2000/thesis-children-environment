# Code for the Final Master Proyect
:uk:
R code used for the Master's Thesis, divided by topic. Data is not provided but may be obtained from the UK Data Service [here](http://doi.org/10.5255/UKDA-SN-9175-3).
1. **Preprocessing**: Selection of rows and columns, recoding of variables and creation of new variables (see Section 2.2). 
2. **Indicators**: Computation of the seven indicators (see Section 2.3).
3. **Clusters**: Computation of distances, selection of the number of clusters and application of the k-medoids algorithm (see Section 2.4).

:es:
Códigos de R utilizados para el TFM, organizados según tema. El conjunto de datos no se provee en este repositorio pero puede descargarse desde el sitio de UK Data Service [aquí](http://doi.org/10.5255/UKDA-SN-9175-3).
1. **Preprocessing**: Selección de filas y columnas, recodificación de variables y creación de nuevas variables (ver Sección 2.2).
2. **Indicators**: Construcción de los siete indicadores (ver Sección 2.3).
3. **Clusters**: Cómputo de las distancias, selección del número de clústers y aplicación del algoritmo k-medoids (ver Sección 2.4).
